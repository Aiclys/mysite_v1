Linux and BSD resources:

The ArchWiki is one of the best places to learn about Arch Linux and its many features. Although it's called ArchWiki alot of the information on
there can be applied to other Linux distributions which makes this resource very handy to have. Make sure to read the FAQ carefully
as most questions will almost certainly be solved.
The Forum is also a great place to talk to other Arch Linux users around the world.

The Gentoo Wiki is a great place to learn about Gentoo specific things. It is also very useful as alot of the information isnÂ´t unique to Gentoo and can be applied to other distributions aswell.

linux.die.net (Linux related documentation):
linux.die.net is a great place to find beginner-friendly documentation related to Linux and its environment. It's also great for advanced users as there are many advanced tutorials on system administration, networking, bash scripting and more.

TLDP (The Linux Documentation Project):
TLDP is one of if not the best place to find advanced and complete documentation about Linux and its ecosystem. It has the most extensive documentation you can find online. Although it is best suited for more advanced users, it can still be practical for beginners who are looking for complete documentation.

GNU Manuals:
The GNU Manuals are a great place to find on- and offline manuals about programming, Linux and free and open source software. The manuals are very extensive and a good read for advanced users and beginners.

Linux Journey:
The Linux Jorney website is a great place for new Linux users or people who consider switching to Linux from Windows. There are alot of beginner-friendly exercizes and it's very simple to get started. The tutorials are also very easy to understand and it's a great introduction to Linux.

FreeBSD Handbook:
The FreeBSD Handbook is probably the best, most complete documentation there is for an operating system. It is very extensive and easy to understand. If you want to get started with BSD or FreeBSD in particular, then take a look at it as it covers pretty much everything there is.

OpenBSD Handbook:
The OpenBSD Handbook is very good aswell. Although it isn't as extensive as the FreeBSD Handbook, it is still informative. For more command-specific information, take a look at the OpenBSD Man Pages.

BSD for Linux users:
BSD for Linux users is a great blog/essay that talks about the main differences between BSD and Linux. It contains a ton of information and is written for users who already have some knowledge of Linux. If you cant decide between BSD and Linux, this essay is the perfect read for you
